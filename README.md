## ğŸš¦ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm, yarn, pnpm, or bun

### Installation

```bash
npm install
# or
yarn install
# or
pnpm install
```

### Development

Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

The site will be available at `http://localhost:3000`

### Build

Build for production:

```bash
npm run build
```

### Start Production Server

```bash
npm start
```
