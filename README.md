# Sabiq Sabry - Portfolio

This is my personal portfolio website showcasing my work, skills, and experience as an AI Engineer and Full-Stack Developer.

## 🚀 Live Site

Visit the portfolio at: [sabiq.vercel.app](https://sabiq.vercel.app)

## 🛠️ Technology Stack

This portfolio is built with:

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Beautiful, accessible component library
- **Framer Motion** - Smooth animations
- **Next Themes** - Dark mode support

## 🚦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm, yarn, pnpm, or bun

### Installation

```bash
npm install
# or
yarn install
# or
pnpm install
```

### Development

Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

The site will be available at `http://localhost:3000`

### Build

Build for production:

```bash
npm run build
```

### Start Production Server

```bash
npm start
```
